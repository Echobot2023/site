O projeto Echo Bot é um projeto de desenvolvimento de um robô assistente virtual. O projeto foi desenvolvido por alunos do curso técnico de informática da Escola Ulbra São Lucas.

O objetivo do projeto é aprimorar as funcionalidades do robô, tornando-o capaz de interagir com as pessoas de forma mais natural e intuitiva. Para isso, estamos desenvolvendo um mecanismo que permita ao robô entender e responder a comandos de voz.

Além disso, estamos aprimorando a aparência do robô, tornando-o mais agradável e atraente. Para isso, estamos trabalhando com designers e artistas para criar uma nova estética para o robô.

O projeto está dividido em três grupos:

  • Grupo de desenvolvimento web: responsável pelo desenvolvimento do website do projeto.
  • Grupo de designers: responsável pelo design do robô.
  • Grupo da mídia: responsável pela divulgação do projeto.

O projeto Echo Bot é um projeto inovador que tem o potencial de revolucionar a forma como interagimos com os robôs. Estamos trabalhando duro para tornar o projeto um sucesso e estamos ansiosos para compartilhar nossos resultados com o mundo.
